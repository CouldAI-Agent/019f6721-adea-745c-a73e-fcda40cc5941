# Business Add App

A clean, responsive Flutter application for registering and adding new business profiles. 

## Features
- **Responsive Form**: Adapts smoothly across mobile, tablet, desktop, and web views.
- **Form Validation**: Client-side validation for emails, phone numbers, and required fields to ensure high data quality.
- **Clean UI**: Uses modern Material 3 design principles for intuitive user interactions.
- **Cross-Platform**: Built to run natively on iOS, Android, macOS, Windows, Linux, and modern Web browsers.

## Main Flows
- **Register Business**: The user navigates to the primary screen, fills in the business details (Name, Email, Phone, Address), and hits Save. Invalid or missing data provides immediate feedback.
- **Feedback Loop**: A snackbar confirms successful mock submissions, and the form resets.

## Tech Stack
- Framework: Flutter
- UI Design: Material 3

## Setup Instructions
1. Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2. Clone this repository.
3. Fetch dependencies: `flutter pub get`
4. Run the app: `flutter run`
